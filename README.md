# УКРАЇНЦІ — фотопроєкт (презентація)

Анімована презентація-відео фотопроєкту «Українці» (Фонд МКН): титульний кадр,
вступ, команда, чотири сезони з фотосеріями та музикою. Працює у браузері на
React + Babel, кадри керуються рушієм анімації `SceneStage`.

## Запуск

JSX-модулі підвантажуються через `fetch`, тому проєкт треба відкривати **через
локальний сервер**, а не подвійним кліком (file:// блокує завантаження).

```bash
./start.sh          # порт 8000, сам відкриє браузер
./start.sh 9000     # інший порт
# або вручну: python3 -m http.server 8000  →  http://localhost:8000/
```

Керування (кнопки в правому нижньому куті):
- **Тема** — вибір оформлення (Класика / Модерн / Архів); змінюється миттєво,
  вибір запамʼятовується для наступного відкриття.
- **Звук** — вмикає/вимикає музичний супровід.
- Пробіл — пауза/відтворення.

## Збірка одним файлом

Щоб отримати **самодостатній HTML** (усе вкладено: бібліотеки, шрифти, фото,
музика, скомпільовані сцени) — його можна відкрити подвійним кліком або
переслати одним файлом, без сервера й без інтернету:

```bash
python3 build.py                       # -> dist/Українці презентація.html
python3 build.py вихід.html            # своя назва/шлях
```

Скрипт `build.py` вкладає всі ассети як data-URI (фото й музику — через механізм
`window.__resources`, тому `config.js` не змінюється), а JSX-модулі загортає в
`<script type="text/babel">`, щоб Babel скомпілював їх у браузері. Залежностей
немає — потрібен лише Python 3. Результат (`dist/`) не потрапляє в git.

## Теми оформлення

Три «характери» презентації (той самий контент і фото):

| Тема | Палітра | Заголовки | Особливість |
|---|---|---|---|
| **Класика** | тепла земляна (беж/теракота) | Playfair Display | мʼякі повільні переходи |
| **Модерн** | холодна ч/б + синій акцент | Montserrat (гротеск) | чіткіші швидші переходи |
| **Архів** | тепла сепія | PT Mono (моноширинний) | зерно + віньєтка + сепія на фото |

Теми описано даними у `src/themes.js` — палітра, шрифти, параметри руху й ефекти.
Щоб додати/змінити тему, редагуйте цей файл (перша в списку — типова).

## Структура

```
index.html              — точка входу (шрифти, бібліотеки, boot-завантажувач)
start.sh                — запуск локального сервера + відкриття браузера
src/
  themes.js             — ТЕМИ оформлення (палітра, шрифти, рух, ефекти)
  config.js             — КОНТЕНТ: фото по сезонах, порядок і тривалість кадрів,
                          команда, вступний текст, музика
  scenes.jsx            — сцени/кадри (логіка, композиція, кнопка вибору теми)
  animations-v2.jsx     — рушій анімації (SceneStage, таймлайн)
  tweaks-panel.jsx      — панель редактора (у standalone не використовується)
assets/
  js/                   — react.js, react-dom.js, babel.min.js
  fonts/                — локальні шрифти (PT Sans, PT Mono, Playfair) + fonts.css
  photos/{winter,spring,summer,autumn}/ — фото по сезонах (1920px, з photo-final)
  audio/soundtrack.mp3  — музичний супровід
photo-final/            — вихідні фото (оригінали, джерело для assets/photos)
Українці презентація v1.html — попередня версія одним файлом (архів/резерв)
```

## Як редагувати

- **Тексти, порядок і тривалість кадрів, список фото** — `src/config.js`.
- **Композиція та анімація кадрів** — `src/scenes.jsx`.
- **Замінити фото сезону** — покладіть файли у `assets/photos/<сезон>/` та
  оновіть відповідний список у `config.js`. Рекомендований розмір — до 1920px,
  JPEG ~80% (як зроблено з `photo-final`).

## Що ще треба від замовника

1. **Другий кадр (вступ):** зараз стоїть **чорновий текст** у `window.INTRO`
   (`config.js`) — замініть на фінальний опис проєкту. Композицію («дихаючі»
   поля, кегль, інтерліньяж) уже налаштовано.
2. **Кадр «Учасники» (фото учасників, перед зимою):** каркас готовий
   (`Participants` у `scenes.jsx`), але **немає самих фото**. Додайте їх у
   `assets/photos/team/`, заповніть `window.PARTICIPANTS` і розкоментуйте кадр
   `Учасники` у `window.OM_SCENES` (`config.js`).

## Враховані зауваження (Anna Lebedeva)

- Кадр 1: «фотопроєкт» піднято вище та збільшено кегль (~1.5×), новий правопис
  «фотопро**Є**кт»; назву **УКРАЇНЦІ** тепер показано літера-за-літерою, як титри команди.
- Кадр 2 (вступ): додано, з «дихаючими» полями та читомим кеглем/інтерліньяжем.
- Кадр 3 (команда): блок із рисочкою піднято вище; літери в «команда проєкту»
  та в посадах трохи наведено (жирніше), зі збереженням легкості; правопис «проєкту».
- Кадр 4 (учасники): підготовлено каркас (потребує фото — див. вище).
- Фото: показуються **на повний екран без білої рамки**; збережено анімацію
  Ken Burns (кадр у сталому форматі динамічно розширюється/звужується).
- Додано **весняну** фотосерію (у попередній версії була заглушка).
